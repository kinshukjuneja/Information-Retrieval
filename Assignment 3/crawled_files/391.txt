(<b>\xc2\xa3</b>) is the <a href="/wiki/Currency_symbol" title="Currency symbol">symbol</a> for the <a href="/wiki/Pound_sterling" title="Pound sterling">pound sterling</a>\xe2\x80\x94the <a href="/wiki/Currency" title="Currency">currency</a> of the <a href="/wiki/United_Kingdom" title="United Kingdom">United Kingdom</a> and previously of <a href="/wiki/Kingdom_of_Great_Britain" title="Kingdom of Great Britain">Great Britain</a> and the <a href="/wiki/Kingdom_of_England" title="Kingdom of England">Kingdom of England</a>. The same symbol is used for similarly named currencies, such as the <a href="/wiki/Gibraltar_pound" title="Gibraltar pound">Gibraltar pound</a>, the <a href="/wiki/Egyptian_pound" title="Egyptian pound">Egyptian pound</a>, the <a href="/wiki/Syrian_pound" title="Syrian pound">Syrian pound</a>, etc. It is also sometimes used for currencies named <i><a href="/wiki/Lira" title="Lira">lira</a></i>, for example the now withdrawn <a href="/wiki/Italian_lira" title="Italian lira">Italian lira</a>.\n</p><p>The symbol derives from a capital "<a href="/wiki/L" title="L">L</a>", representing <i>libra pondo</i>, the basic unit of weight in the <a href="/wiki/Roman_Empire" title="Roman Empire">Roman Empire</a>, which in turn is derived from the <a href="/wiki/Latin" title="Latin">Latin</a> word, <i>libra</i>, meaning <a href="/wiki/Weighing_scale" title="Weighing scale">scales</a> or a balance. The pound became an <a href="/wiki/Kingdom_of_England" title="Kingdom of England">English</a> unit of weight and in England became defined as the <a href="/wiki/Pound_(weight)#Tower_pound" class="mw-redirect" title="Pound (weight)">tower pound</a> (equivalent to 350 grams) of fine (pure) <a href="/wiki/Silver" title="Silver">silver</a>. According to the <a href="/wiki/Royal_Mint_Museum" title="Royal Mint Museum">Royal Mint Museum</a>:\n</p>\n<style data-mw-deduplicate="TemplateStyles:r856303468">.mw-parser-output .templatequote{overflow:hidden;margin:1em 0;padding:0 40px}.mw-parser-output .templatequote .templatequotecite{line-height:1.5em;text-align:left;padding-left:1.6em;margin-top:0}</style><blockquote class="templatequote"><p>It is not known for certain when the horizontal line or lines, which indicate an abbreviation, first came to be drawn through the L. However, there is in the <a href="/wiki/Bank_of_England_Museum" title="Bank of England Museum">Bank of England Museum</a> a cheque dated 7 January 1661 with a clearly discernible \xc2\xa3 sign. By the time the Bank was founded in 1694 the \xc2\xa3 sign was in common use.<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup>\n</p></blockquote>\n<p>However, the simple letter L, in lower- or uppercase, was used to represent the pound sterling in printed books and newspapers until well into the 19th century.\n</p><p>The pound sign is placed before the number (e.g. "\xc2\xa312,000") and <a href="/wiki/Space_(punctuation)" title="Space (punctuation)">separated</a> from the following digits by no space or only a <a href="/wiki/Thin_space" title="Thin space">thin space</a>.\n</p><p>The symbol \xe2\x82\xa4 (note the double dash at its middle) was called the <b>lira sign</b> in <a href="/wiki/Italy" title="Italy">Italy</a>, before the adoption of the <a href="/wiki/Euro" title="Euro">euro</a>. It was used (in free variation with \xc2\xa3) as an alternative to the more usual <b>L.</b> or <b>Lit.</b> to show prices in <a href="/wiki/Italian_lira" title="Italian lira">lire</a>. It was also used unofficially as the symbol of the <a href="/wiki/Maltese_lira" title="Maltese lira">Maltese lira</a> instead of the official Lm.\n</p><p>In American English, the term "pound sign" usually refers to the symbol <a href="/wiki/Number_sign" title="Number sign">#</a>, and the corresponding telephone key is called the "pound key".<sup id="cite_ref-nyt_2-0" class="reference"><a href="#cite_note-nyt-2">&#91;2&#93;</a></sup> In Canadian English the symbols \xc2\xa3 and # are both called the pound sign, but the # is also referred to as the "<a href="/wiki/Number_sign" title="Number sign">number sign</a>" and the "noughts-and-crosses board".<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup>\n</p><p>In the eighteenth-century <a href="/wiki/Caslon" title="Caslon">Caslon</a> metal fonts, the pound sign was identical to the italic capital "<a href="/wiki/J" title="J">J</a>" rotated 180 degrees.<sup id="cite_ref-Caslon&#39;s_punches_and_matrices_Howes_4-0" class="reference"><a href="#cite_note-Caslon&#39;s_punches_and_matrices_Howes-4">&#91;4&#93;</a></sup>\n</p>\n<div id="toc" class="toc"><input font-size: smaller;">Lira sign</span> (HTML&#xa0;<code>&amp;#8356;</code>)<sup id="cite_ref-unicode2_6-0" class="reference"><a href="#cite_note-unicode2-6">&#91;6&#93;</a></sup> Unicode notes that the "lira sign" is not widely used and was added due to both it and the pound sign being available on <a href="/wiki/HP_Roman" title="HP Roman">HP printers</a> and also claim that the preferred sign for lira is the pound sign.<sup id="cite_ref-Unicode_80_Lira_7-0" class="reference"><a href="#cite_note-Unicode_80_Lira-7">&#91;7&#93;</a></sup>\n</p><p>Some fonts render the pound sign with a double bar (for example on a previous version of the British five pound note);<sup id="cite_ref-oldenglishbanknotes_8-0" class="reference"><a href="#cite_note-oldenglishbanknotes-8">&#91;8&#93;</a></sup> this is simply a different <a href="/wiki/Glyph" class="mw-redirect" title="Glyph">glyph</a>, and the underlying character (and therefore Unicode code point) is still a pound sign, and not a lira sign.\n</p><p>The encoding of the \xc2\xa3 symbol in position xA3 was standardised by <a href="/wiki/Latin-1" class="mw-redirect" title="Latin-1">Latin-1</a> in 1985. Position xA3 was used by the <a href="/wiki/Digital_Equipment_Corporation" title="Digital Equipment Corporation">Digital Equipment Corporation</a> <a href="/wiki/VT220" title="VT220">VT220</a> terminal, the Amstrad CPC, the Commodore Amiga and the Acorn Archimedes. The IBM PC originally used a non-standard 8-bit character set <a href="/wiki/Code_page_437" title="Code page 437">Code page 437</a> in which the \xc2\xa3 symbol was encoded as x9C; adoption of ISO character codes only came later with <a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a>. The Atari ST also used position x9C. The HP Laserjet used position xBA for the \xc2\xa3 symbol, while most other printers used x9C.\n</p><p>The BBC <a href="/wiki/Ceefax" title="Ceefax">Ceefax</a> system which dated from 1976 encoded the \xc2\xa3 as x23. Many early computers used a variant of <a href="/wiki/ASCII" title="ASCII">ASCII</a> with one of the less-frequently used characters replaced with the \xc2\xa3. The UK national variant of <a href="/wiki/ISO_646" class="mw-redirect" title="ISO 646">ISO 646</a> was standardised as BS 4730 in 1985. This code was identical to ASCII except for two characters: x23 encoded "\xc2\xa3" instead of "#", while x7E encoded "\xe2\x80\xbe" instead of "~". The ZX Spectrum and the BBC Micro used x60 ("`"). The Commodore 64 used x5C ("\\") and the Oric used x5F ("_").\n</p><p>IBM\'s EBCDIC code page <a href="/wiki/EBCDIC_037" title="EBCDIC 037">037</a> uses xB1 for the \xc2\xa3 while code page <a href="/wiki/EBCDIC_285" title="EBCDIC 285">285</a> uses x5B. <a href="/wiki/International_Computers_Limited" title="International Computers Limited">ICL</a>\'s <a href="/wiki/ICT_1900_series" title="ICT 1900 series">1900-series</a> mainframes used a six-bit encoding for characters, loosely based on BS 4730, with the \xc2\xa3 symbol represented as <i>octal</i> 23 (hex 13).\n</p>\n<h2><span
