(concurrency library)">Joins Concurrency Library</a></td></tr></tbody></table>\n<p><b>C\xcf\x89</b> (pronounced "cee <a href="/wiki/Omega" title="Omega">omega</a>";<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">&#91;1&#93;</a></sup> usually written "Cw" or "Comega" whenever the "\xcf\x89" symbol is not available) is a free extension to the <a href="/wiki/C_Sharp_(programming_language)" title="C Sharp (programming language)">C#</a> <a href="/wiki/Programming_language" title="Programming language">programming language</a>, developed by the WebData team in <a href="/wiki/Microsoft_SQL_Server" title="Microsoft SQL Server">Microsoft SQL Server</a> in collaboration with <a href="/wiki/Microsoft_Research" title="Microsoft Research">Microsoft Research</a> in the <a href="/wiki/United_Kingdom" title="United Kingdom">UK</a> and Redmond.  It was formerly known as the <a href="/wiki/Codename" class="mw-redirect" title="Codename">codenames</a> <b>X#</b> (X Sharp) and <b>Xen</b>. It was renamed C\xcf\x89 after <a href="/wiki/Polyphonic_C_sharp" class="mw-redirect" title="Polyphonic C sharp">Polyphonic C#</a>, another research language based on the <a href="/wiki/Join_calculus" class="mw-redirect" title="Join calculus">join calculus</a>, was integrated into it.\n</p>\n<h2><span class="mw-headline" id="Details">Details</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=C%CF%89&amp;action=edit&amp;section=1" title="Edit section: Details">edit</a><span class="mw-editsection-bracket">]</span></span></h2>\n<p>C\xcf\x89 attempts to make datastores (such as <a href="/wiki/Databases" class="mw-redirect" title="Databases">databases</a> and <a href="/wiki/XML" title="XML">XML</a> documents) accessible with the same ease and <a href="/wiki/Type_safety" title="Type safety">type safety</a> as traditional types like <a href="/wiki/String_(computer_science)" title="String (computer science)">strings</a> and <a href="/wiki/Array_data_type" title="Array data type">arrays</a>. Many of these ideas were inherited from an earlier incubation project within the WebData XML team called X# and Xen. C\xcf\x89 also includes new constructs to support <a href="/wiki/Concurrent_programming" class="mw-redirect" title="Concurrent programming">concurrent programming</a>; these features were largely derived from the earlier <a href="/wiki/Polyphonic_C_sharp" class="mw-redirect" title="Polyphonic C sharp">Polyphonic C#</a> project.\n</p><p>Available now as a compiler preview, C\xcf\x89\'s features have been used in the creation of the <a href="/wiki/Language_Integrated_Query" title="Language Integrated Query">LINQ</a> extensions in C#.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">&#91;2&#93;</a></sup> The concurrency constructs have also been released in a slightly modified form as a library, named <a href="/wiki/Joins_(concurrency_library)" title="Joins (concurrency library)"><i>Joins Concurrency Library</i></a>, for <a href="/wiki/C_Sharp_(programming_language)" title="C Sharp (programming language)">C#</a> and other .NET languages by <a href="/wiki/Microsoft_Research" title="Microsoft Research">Microsoft Research</a>.<sup id="cite_ref-3" class="reference"><a href="#cite_note-3">&#91;3&#93;</a></sup>\n</p>\n<h2><span class="mw-headline" id="References">References</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=C%CF%89&amp;action=edit&amp;section=2" title="Edit section: References">edit</a><span class="mw-editsection-bracket">]</span></span></h2>\n<div class="reflist" style="list-style-type: decimal;">\n<div class="mw-references-wrap"><ol class="references">\n<li id="cite_note-1"><span class="mw-cite-backlink"><b><a href="#cite_ref-1">^</a></b></span> <span class="reference-text"><cite class="citation web"><a rel="nofollow" class="external text" href="http://research.microsoft.com/Comega/">"Comega Home"</a>. <i>Microsoft Research</i><span class="reference-accessdate">.
